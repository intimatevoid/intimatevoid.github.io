Arma 3

Videos and scripts

I play Arma 3 with a unit called ANZIF. I post videos of some of our operations on my youtube page. I'll embed some videos later.

I also make scripts! My current offerings are:

  • civKilled.sqf, which tracks civilian deaths, and allows you to run code on players who killed them. Example code names and shames them.
  • countdown.sqf, which displayers a countdown timer in minutes and seconds using the hint system.
  • gearRandomiser.sqf, which allows you to randomise aspects of player equipment at mission start.
  • missionTitles.sqf, a variation on Alias' mission titles script, that prints your time, date, location, and then the mission title and subtitle.
  • rearm.sqf, which allows you to add a scroll wheel option to a crate, from which players can immediately rearm a loadout automatically based on their role.

All are WIPs! Make sure you test them thoroughly on your own server before running them live! I take no responsibility for you messing up your mission!
